# ssh-manager
easy to use ssh by password. (windows coming soon...)

## usage

```bash
chmod +x ./sshm.sh
```

### global use

```bash
cp sshm.sh /usr/local/bin/sshm
# use just like:
sshp ls
```

### add
```bash
sshm add
```

### list
```bash
sshm ls
```

### login server
```bash
sshm server_name
```

### delete
```bash
sshm rm server_name
```
